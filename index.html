About Me

Hi. I'm Çağatay Sel. I'm a senior year computer science student at Bilkent University and also a candidate engineer at HAVELSAN Inc. I have been a computer enthusiast i got my first Windows 97 computer. Today, I'm pursuing a degree in the field that i love and hoping to keep working in the field of computer technologies after i graduate.

Computer science is something more for me than just getting a bachelors degree. I love designing and coding systems that can make people's lives easier. Therefore, i spend lots of time working on developing my side projects that teach me many things firsthand.

In this site, you can find the projects that i have developed since I started my education in Bilkent University.

Used Technologies

Android

I have done an android project in my first year for CS 102 - Algorithms and Programming II course. After the course, I have kept developing the application for over a year and published my first application on Google Play Store in 2017. You can find more information about this application here.

I have also worked with a group on a multiplayer android game for CS 319 Object-Oriented Software Engineering course. I have worked on both server and client sides of this application. You can also see the details of this application here.

WebVR & WebGL

During my internship and part time work in HAVELSAN, I have developed an VR web application called IFE(in-flight-entertainment) VR. For -this application, I have used Cesium library which was using WebGL. In addition to Cesium, I have also utilized A-frame framework and Cesium-vr for VR functionalities.

Web Design

I have designed a website from scratch as part of GRA 210 - Web Design course. For this project, I have used Adobe Photoshop; HTML,CSS and JavaScript while designing my the site. More information about this project can be found here.

Besides my project, i'm also developing the site you are currently on during my free times. I'm currently using Bootstrap and jQuery libraries and hoping to add more libraries for new features.

Server - Client Architecture

As part of CS319 - Object Oriented Software Engineering course's term project, I have coded the server side of a multiplayer android game by using Java Sockets on both sides. I have also coded a facade class on android for establishing connection to server in order to offer an interface to other team members working on client side. During this project, i have used multithreading and had firsthand experience with concepts such as concurrency and synchronization. More information about this application can be found here.

FPGA's and Digital Design

For CS223 - Digital Design course, i have worked with a friend to develop an obstacle race game by using Basys3 FPGA and a 8x8 dot matrix. We have used System Verilog as our hardware description language (HDL). More information about this application can be found here. Besides our project, I have also done lab tutoring at Bilkent University for Digital Design course for 2 semesters.

Adobe Photoshop

I have used photoshop for my web design course's project in order to go beyond fixed templates of websites and actually design a page from a designers perspective. I've also used photoshop for photo manipulation and for increasing photograph qualities. I also enjoy colorizing old war photos in my free time. You can have look at to some of my work here.

Used Programming Languages

  • Java: I have been using java for 3 years now and I have written 2 android applications and backend side of an multiplayer application using java.
  • JavaScript: I have used JavaScript while developing 2 web applications:IFE VR and Patient Tracking System. I have also utilized JavaScript while designing 2 websites.
  • HTML / CSS: I have been using these guys for web designing of course.
  • C++: I have used C++ for a year and done many programming assignments that involved analysis of sorting algorithms and usage of data structures such as lists, stacks, queues, hashes, binary search trees, balanced search trees,heaps etc.
  • PHP: I have used PHP for the backend side of our Database Systems project.
  • C: I have used C language during CS342 - Operating Systems course. I have done 4 small projects which included developing multi processed/threaded programs, kernel modules for Ubuntu.
